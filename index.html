Bank Service


This project represents simple bank system that displays bank transactions such as payments and withdrawals for a given user.

During account registration its required to provide the initial account balance.
Deposit and withdraw logic works on provided account balance. To read data from specific account only ID account is required.

Status of the transaction have two options: Accepted or declined. Created users and all information are saved in mocked database.
Here, I have learnt how to create unit tests and how to test applications using Swagger UI.
